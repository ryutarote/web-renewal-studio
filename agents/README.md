# エージェント定義

各エージェントのプロンプトとロジックを管理するディレクトリ。

## エージェント一覧

| ファイル | エージェント | 役割 |
|---|---|---|
| `researcher.md` | リサーチャー | YouTube・X・Instagram巡回、ネタ収集 |
| `writer.md` | ライター | 投稿生成（15パターン＋品質採点） |
| `poster.md` | ポスター | API経由で時間帯分散投稿 |
| `fetcher.md` | フェッチャー | 投稿メトリクス自動取得 |
| `analyst.md` | アナリスト | パフォーマンス分析・改善提案 |
